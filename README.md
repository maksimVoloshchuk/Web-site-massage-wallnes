# LINIEN · Massage Aesthetic

Премиальный лендинг для студии массажа **LINIEN Massage Aesthetic** — выполнен в элегантной SPA-эстетике с бежево-каменной палитрой, serif-типографикой и минималистичной премиальной композицией.

---

## ✅ Реализованные функции

- **Hero-секция** — полноэкранный заголовок с эффектом параллакса, декоративными драпировками и анимацией scroll-hint
- **О студии** — текстовый блок с декоративными камнями, свечой с CSS-анимацией и счётчиками статистики
- **Услуги и цены** — 4 карточки со всеми услугами, длительностями и ценами из оригинального прайс-листа
- **Преимущества** — 6 блоков преимуществ студии
- **Атмосфера** — тёмная SPA-секция с декоративными элементами (камни, свеча, лотос)
- **Призыв к записи (CTA)** — секция с декоративным оформлением и кнопками
- **Контакты и форма** — форма записи со всеми услугами в select, сохранение заявок через Table API
- **Footer** — логотип, навигация, слоган

### Интерактивность
- Фиксированная навигация с эффектом при прокрутке
- Мобильное меню-бургер
- Плавная прокрутка к якорям
- Scroll-анимации появления элементов (IntersectionObserver)
- Параллакс hero-секции
- Анимация счётчиков статистики
- Flickering CSS-анимация свечи
- Валидация и отправка формы с сохранением в БД
- Кнопка «наверх»

## 🗄 База данных

**Таблица: `inquiries`** — хранит заявки на запись
| Поле    | Тип       | Описание |
|---------|-----------|----------|
| id      | text      | UUID |
| name    | text      | Имя клиента |
| phone   | text      | Телефон / WhatsApp |
| service | text      | Выбранная услуга |
| message | rich_text | Сообщение (опционально) |
| date    | datetime  | Дата отправки |

**API endpoints:**
- `GET  tables/inquiries` — список заявок
- `POST tables/inquiries` — создать заявку (отправка формы)
